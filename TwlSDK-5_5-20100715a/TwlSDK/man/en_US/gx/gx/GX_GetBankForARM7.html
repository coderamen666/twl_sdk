GX_GetBankForARM7

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamARM7 GX_GetBankForARM7(void);

Arguments

None.

Return Values

The settings of the VRAM bank allocated to the ARM7 memory space.

Description

This function returns the settings of the VRAM bank allocated to ARM7. Refer to the GX_SetBankForARM7 function for details on the GXVRamARM7 type.

See Also

GX_SetBankForARM7, GX_ResetBankForARM7, GX_DisableBankForARM7
