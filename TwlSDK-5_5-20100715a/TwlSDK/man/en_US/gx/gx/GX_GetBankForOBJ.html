GX_GetBankForOBJ

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamOBJ GX_GetBankForOBJ(void);

Arguments

None.

Return Values

The settings of the VRAM bank allocated to the main 2D Engine's OBJ.

Description

This function returns the settings of the VRAM bank allocated to the Main 2D Engine OBJ. Refer to the GX_SetBankForOBJ function for details on the GXVRamOBJ type.

See Also

GX_SetBankForOBJ, GX_ResetBankForOBJ, GX_DisableBankForOBJ
