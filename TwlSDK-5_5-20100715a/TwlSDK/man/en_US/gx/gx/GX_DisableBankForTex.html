GX_DisableBankForTex

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamTex GX_DisableBankForTex(void);

Arguments

None.

Return Values

The VRAM bank allocated to the texture image slot.

Description

This function sets the VRAM bank allocated to the texture image slot. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that have been allocated to each resource will not be disabled unless disabled with this function. Refer to the GX_SetBankForTex function for details on the GXVRamTex type.

See Also

GX_SetBankForTex, GX_GetBankForTex, GX_ResetBankForTex
