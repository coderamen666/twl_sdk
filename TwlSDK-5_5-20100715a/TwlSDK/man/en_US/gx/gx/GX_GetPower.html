GX_GetPower

Syntax

#include <nitro/gx/gx.h>

GXPower GX_GetPower(void);

Arguments

None.

Return Values

Status of each type of power control.

Description

This function acquires the status of each type of power control. Refer to the GX_SetPower for details on the GXPower type.

See Also

GX_SetPower
