GX_GetDispSelect

Syntax

#include <nitro/gx/gx.h>

GXDispSelect GX_GetDispSelect(void);

Arguments

None.

Return Values

LCD output destination setting.

Description

This function gets the LCD output destination settings. Refer to the GX_SetDispSelect function for details on the GXDispSelect type.

See Also
