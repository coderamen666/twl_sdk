GX_GetBankForSubBG

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubBG GX_GetBankForSubBG(void);

Arguments

None.

Return Values

The settings of the VRAM bank allocated to the sub 2D Engine's BG.

Description

This function returns settings of the VRAM bank allocated to the sub 2D Engine BG. Refer to the GX_SetBankForSubBG function for details on the GXVRamSubBG type.

See Also

GX_SetBankForSubBG, GX_ResetBankForSubBG, GX_DisableBankForSubBG
