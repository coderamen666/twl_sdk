GX_GetBankForSubOBJ

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubOBJ GX_GetBankForSubOBJ(void);

Arguments

None.

Return Values

The settings of the VRAM bank allocated to the sub 2D Engine's OBJ.

Description

This function returns the settings of the VRAM bank allocated to the sub 2D Engine OBJ. Refer to the GX_SetBankForSubOBJ function for details on the GXVRamSubOBJ type.

See Also

GX_SetBankForSubOBJ, GX_ResetBankForSubOBJ, GX_DisableBankForSubOBJ
