GX_DisableBankForSubOBJ

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubOBJ GX_DisableBankForSubOBJ(void);

Arguments

None.

Return Values

The VRAM bank allocated to the sub 2D Engine's OBJ.

Description

This function disables the VRAM bank allocated to the sub 2D Engine OBJ. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. Refer to the GX_SetBankForSubOBJ function for details on the GXVRamSubOBJ type.

See Also

GX_SetBankForSubOBJ, GX_GetBankForSubOBJ, GX_ResetBankForSubOBJ
