GX_DisableBankForOBJ

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamOBJ GX_DisableBankForOBJ(void);

Arguments

None.

Return Values

The VRAM bank that is allocated to the main 2D Engine's OBJ.

Description

This function disables the VRAM bank that is allocated to the Main 2D Engine OBJ. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. Refer to the GX_SetBankForOBJ function for details on the GXVRamOBJ type.

See Also

GX_SetBankForOBJ, GX_GetBankForOBJ, GX_ResetBankForOBJ
