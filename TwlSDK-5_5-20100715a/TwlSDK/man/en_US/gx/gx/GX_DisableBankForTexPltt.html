GX_DisableBankForTexPltt

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamTexPltt GX_DisableBankForTexPltt(void);

Arguments

None.

Return Values

The VRAM bank allocated to the texture palette slot.

Description

This function sets the VRAM bank allocated to the texture palette slot. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that were allocated to each resource will not be disabled unless disabled with this function. Refer to the GX_SetBankForTexPltt function for details on the GXVRamTexPltt type.

See Also

GX_SetBankForTexPltt, GX_GetBankForTexPltt, GX_ResetBankForTexPltt
