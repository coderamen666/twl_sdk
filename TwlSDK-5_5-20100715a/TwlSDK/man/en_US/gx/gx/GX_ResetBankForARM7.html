GX_ResetBankForARM7

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamARM7 GX_ResetBankForARM7(void);

Arguments

None.

Return Values

VRAM bank allocated to the ARM7 memory space.

Description

This function allocates to the LCDC memory space the VRAM bank that was allocated to the ARM7. Refer to the GX_SetBankForARM7 function for details on the GXVRamARM7 type.

See Also

GX_SetBankForARM7, GX_GetBankForARM7, GX_DisableBankForARM7
