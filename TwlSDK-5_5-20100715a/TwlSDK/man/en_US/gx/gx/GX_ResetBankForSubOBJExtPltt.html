GX_ResetBankForSubOBJExtPltt

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamSubOBJExtPltt GX_ResetBankForSubOBJExtPltt(void);

Arguments

None.

Return Values

The VRAM bank that is allocated to the sub 2D Engine's OBJ extended palette.

Description

This function allocates to LCDC the VRAM bank that was allocated to the sub 2D Engine's OBJ extended palette. Refer to the GX_SetBankForSubOBJExtPltt function for details on the GXVRamSubOBJExtPltt type.

See Also

GX_SetBankForSubOBJExtPltt, GX_GetBankForSubOBJExtPltt, GX_DisableBankForSubOBJExtPltt
