GX_GetSizeOfTexPltt

Syntax

#include <nitro/gx/gx_vramcnt.h>

u32 GX_GetSizeOfTexPltt(void);

Arguments

None.

Return Values

Size of texture palette memory.

Description

Returns the number of bytes of memory allocated to the texture palette.
