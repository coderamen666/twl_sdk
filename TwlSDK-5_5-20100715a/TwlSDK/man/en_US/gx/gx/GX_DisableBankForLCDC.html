GX_DisableBankForLCDC

Syntax

#include <nitro/gx/gx_vramcnt.h>

GXVRamLCDC GX_DisableBankForLCDC(void);

Arguments

None.

Return Values

The VRAM bank allocated to the LCDC memory space.

Description

This function disables the VRAM bank allocated to the LCDC memory space. If disabled, VRAM cannot be accessed, but the VRAM bank contents are not lost. In addition, the VRAM banks that have been allocated to each resource will not be disabled unless disabled with this function. Refer to the GX_SetBankForLCDC function for details on the GXVRamLCDC type.

See Also

GX_SetBankForLCDC, GX_GetBankForLCDC
