G3X_ResetMtxStackOverflow

Syntax

#include <nitro/gx/g3x.h>

void G3X_ResetMtxStackOverflow(void);

Arguments

None.

Return Values

None.

Description

This function resets the matrix stack overflow/underflow flag. It is also indirectly called when the G3X_Init, G3X_Reset, G3X_InitMtxStack, or G3X_ResetMtxStack function is called.

See Also

G3X_Init, G3X_Reset, G3X_InitMtxStack, G3X_ResetMtxStack
