G3X_Init

Syntax

#include <nitro/gx/g3x.h>

void G3X_Init();

Arguments

None.

Return Values

None.

Description

The function initializes the 3D-related registers. Calls the G3X_InitMtxStack and G3X_InitTable functions internally.

See Also

GX_Init, G3X_Reset, G3X_InitTable, G3X_InitMtxStack
