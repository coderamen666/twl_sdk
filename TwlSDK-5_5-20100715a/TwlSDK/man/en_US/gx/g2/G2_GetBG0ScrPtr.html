G2_GetBG0ScrPtr

Syntax

#include <nitro/gx/g2.h>

void *G2_GetBG0ScrPtr();

Arguments

None.

Return Values

The pointer to the region storing the BG0 screen.

Description

This function returns the pointer to the region where the Main 2D Engine BG0 screen data is stored. The return values are as follows:

HW_BG_VRAM
+ (Value at the end of the name in the GXBGScrOffset enumerated type specified by GX_SetBGScrOffset)
+ (Value at the end of the name in the GXBGScrBase enumerated type specified by G2_SetBG0Control)

See Also

G2_SetBG0Control, G2_SetBG0Priority, G2_BG0Mosaic, G2_GetBG0CharPtr, G2_SetBG0Offset
