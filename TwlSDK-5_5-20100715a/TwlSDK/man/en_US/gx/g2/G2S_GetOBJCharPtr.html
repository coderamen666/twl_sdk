G2S_GetOBJCharPtr

Syntax

#include <nitro/gx/g2.h>

void *G2S_GetOBJCharPtr();

Arguments

None.

Return Values

Pointer to the region where sub 2D Engine's OBJ character data is stored.

Description

This function only returns (void*)HW_DB_OBJ_VRAM.
