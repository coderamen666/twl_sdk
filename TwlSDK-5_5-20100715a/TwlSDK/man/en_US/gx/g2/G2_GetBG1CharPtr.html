G2_GetBG1CharPtr

Syntax

#include <nitro/gx/g2.h>

void *G2_GetBG1CharPtr();

Arguments

None.

Return Values

The pointer to the region where BG1 character data is stored.

Description

This function returns the pointer to the region where the Main 2D Engine BG1 character data is stored. The return values for character BGs are as follows:

HW_BG_VRAM
+ (Value at the end of the name in the GXBGCharOffset enumerated type specified by GX_SetBGCharOffset)
+ (Value at the end of the name in the GXBGCharBase enumerated type specified by G2_SetBG1Control)

See Also

G2_SetBG1Control, G2_SetBG1Priority, G2_BG1Mosaic, G2_GetBG1ScrPtr, G2_SetBG1Offset
