G2_GetBG0CharPtr

Syntax

#include <nitro/gx/g2.h>

void *G2_GetBG0CharPtr();

Arguments

None.

Return Values

The pointer to the region where BG0 character data is stored.

Description

This function returns the pointer to the region where the Main 2D Engine BG0 character data is stored. The return values for character BGs are as follows:

HW_BG_VRAM
+ (Value at the end of the name in the GXBGCharOffset enumerated type specified by GX_SetBGCharOffset)
+ (Value at the end of the name in the GXBGCharBase enumerated type specified by G2_SetBG0Control)

See Also

G2_SetBG0Control, G2_SetBG0Priority, G2_BG0Mosaic, G2_GetBG0ScrPtr, G2_SetBG0Offset
