G2S_SetBG1Offset

Syntax

#include <nitro/gx/g2.h>

void G2S_SetBG1Offset(int hOffset, int vOffset);

Arguments

hOffset Horizontal offset setting
vOffset Vertical offset setting

Return Values

None.

Description

This function specifies the display screen offset for the sub 2D Engine's BG1 screen, in pixels.

See Also

G2S_SetBG0Offset, G2S_SetBG2Offset, G2S_SetBG3Offset
