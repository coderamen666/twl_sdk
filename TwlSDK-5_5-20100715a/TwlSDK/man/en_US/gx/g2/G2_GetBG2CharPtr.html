G2_GetBG2CharPtr

Syntax

#include <nitro/gx/g2.h>

void *G2_GetBG2CharPtr();

Arguments

None.

Return Values

The pointer to the region where BG2 character data is stored.

Description

This function returns the pointer to the region where the Main 2D Engine BG2 character data is stored. The return values for character BGs are as follows:

HW_BG_VRAM
+ (Value at the end of the name in the GXBGCharOffset enumerated type specified by GX_SetBGCharOffset)
+ (Value at the end of the name in the GXBGCharBase enumerated type specified by G2_SetBG2ControlText)

Since there is no character data for bitmap BGs, NULL is returned.

See Also

G2_SetBG2ControlText, ... G2_SetBG2Priority, G2_BG2Mosaic, G2_GetBG2ScrPtr, G2_SetBG2Offset
