MTX_Transpose44

Syntax

#include <nitro/fx/fx_mtx44.h>

void MTX_Transpose44(const MtxFx44 * pSrc, MtxFx44 * pDst);

Arguments

pSrc Pointer to a 4x4 matrix.
pDst Pointer to a 4x4 matrix.

Return Values

None.

Description

*This function sets the transpose matrix of *pSrc in *pDst. pSrc and pDst can be pointers to the same matrix.
