MTX_Copy33To44

Syntax

#include <nitro/fx/fx_mtx33.h>

void MTX_Copy33To44(const MtxFx33 * pSrc, MtxFx44 * pDst);

Arguments

pSrc Pointer to a 3x3 matrix.
pDst Pointer to a 4x4 matrix.

Return Values

None.

Description

This function copies a 3x3 matrix to a 4x4 matrix. The element in the 4th row of the 4th column of the 4x4 matrix is set to FX32_ONE; all other elements in the 4th row and 4th column are set to 0.

See Also

MtxFx33, MtxFx44
