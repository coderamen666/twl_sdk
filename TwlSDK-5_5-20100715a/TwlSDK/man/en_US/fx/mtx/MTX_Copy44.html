MTX_Copy44

Syntax

#include <nitro/fx/fx_mtx44.h>

void MTX_Copy44(const MtxFx44 *pSrc, MtxFx44 *pDst);

Arguments

pSrc Pointer to a 4x4 matrix.
pDst Pointer to a 4x4 matrix.

Return Values

None.

Description

This function copies a 4x4 matrix to a 4x4 matrix.
