MTX_Scale44

Syntax

#include <nitro/fx/fx_mtx44.h>

void MTX_Scale44(MtxFx44 * pDst, fx32 x, fx32 y, fx32 z);

Arguments

pDst Pointer to a 4x4 matrix.
x The scale factor along the x-axis.
y The scale factor along the y-axis.
z The scale factor along the z-axis.

Return Values

None.

Description

*This function sets the scale matrix in *pDst.

See Also

MtxFx44, fx32
