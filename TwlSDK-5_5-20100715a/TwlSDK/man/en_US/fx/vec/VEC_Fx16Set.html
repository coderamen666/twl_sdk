VEC_Fx16Set

Syntax

#include <nitro/fx/fx_vec.h>

void VEC_Fx16Set(
    VecFx16 * a,
    fx16 x, fx16 y, fx16 z
);

Arguments

a Pointer to the vector structure.
x X component.
y Y component.
z None.

Return Values

None.

Description

Sets values to a type VecFx16 vector.
