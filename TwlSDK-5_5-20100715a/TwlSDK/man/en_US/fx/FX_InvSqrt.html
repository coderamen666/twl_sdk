FX_InvSqrt

Syntax

#include <nitro/fx/fx_cp.h>

fx32 FX_InvSqrt(fx32 x);

Arguments

x fx32-format fixed-point decimal.

Return Values

Reciprocal of the square root of x.

Description

This function calculates the reciprocal of an fx32-format square root. The result is returned as an fx32 type. The Divider and Square Root Calculator are used internally. Refer to Divider Usage Notes when using this function inside an interrupt.

See Also

FX_Sqrt, FX_Inv
