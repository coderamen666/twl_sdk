FX_GetInvResultFx64c

Syntax

#include <nitro/fx/fx_cp.h>

fx64c FX_GetInvResultFx64c(void);

Arguments

None.

Return Values

Results of the reciprocal calculations.

Description

Returns the results of calculating the inverse with FX_InvAsync and FX_InvAsyncImm in fx64c format.

See Also

FX_InvAsync, FX_InvAsyncImm, FX_GetInvResult
