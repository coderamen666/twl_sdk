SSP_GetBmpFileSize

Syntax


u32 SSP_GetBmpFileSize(u16 width, u16 height);

Arguments

width Image width. This must be a multiple of 2.
height Image height.

Return Values

Returns the size of a BMP file.

Description

Gets the size of a BMP file created by the SSP_CreateBmpFile function.
