SSP_GetJpegDecoderSoftware

Syntax


u16 SSP_GetJpegDecoderSoftware(u8 *buffer);

Arguments

buffer Buffer used to store the game code of applications that encode JPEG files.

Return Values

Returns 4, the size of the game code.

Description

Gets the value of the application's game code that is encoded. Game codes are obtained when decoding.

See Also

SSP_StartJpegDecoder
