TCLSearchObject

Definition

#include <twl/tcl.h>


typedef struct
{
    int startIdx;
    u32 condition;
} TCLSearchObject;

Description

Use this structure with search-related functions.

It can be used after it is initialized by the TCL_InitSearchObject function.

The direct use of any of its members from an application is prohibited.
