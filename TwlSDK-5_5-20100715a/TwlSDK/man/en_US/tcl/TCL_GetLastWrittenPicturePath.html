TCL_GetLastWrittenPicturePath

Definition

#include <twl/tcl.h>


const char* TCL_GetLastWrittenPicturePath( void );

Arguments

None.

Return Values

Path of the image written last

Description

Returns the path of the last image written by the TCL_EncodeAndWritePicture(Ex) functions.

See Also

TCL_EncodeAndWritePicture(Ex)
