TCL_GetLastWrittenPictureInfo

Definition

#include <twl/tcl.h>


const TCLPictureInfo* TCL_GetLastWrittenPictureInfo( void );

Arguments

None.

Return Values

Photo information for the last image to be written.

Description

Returns photo information for the last image written by the TCL_EncodeAndWritePicture(Ex) functions.
