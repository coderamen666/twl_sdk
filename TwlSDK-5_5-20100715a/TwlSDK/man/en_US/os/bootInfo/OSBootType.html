OSBootType

Definition

#include <nitro/os.h>
typedef u16 OSBootType;
#define    OS_BOOTTYPE_ILLEGAL      0
#define    OS_BOOTTYPE_ROM          1
#define    OS_BOOTTYPE_DOWNLOAD_MB  2
#define    OS_BOOTTYPE_NAND         3 
#define    OS_BOOTTYPE_MEMORY       4

Elements

OS_BOOTTYPE_ILLEGAL Illegal status
OS_BOOTTYPE_ROM Status when booted from a ROM
OS_BOOTTYPE_DOWNLOAD_MB Status when booted from a downloaded application
OS_BOOTTYPE_NAND Status when booted from an application in NAND
OS_BOOTTYPE_MEMORY Status when running only in a static segment at boot time

Description

Constants indicating the boot types for the TWL console.

Revision History

2008/01/25 Changed the typedefs to u16.
