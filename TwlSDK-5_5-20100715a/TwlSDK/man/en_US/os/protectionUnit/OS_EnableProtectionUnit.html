OS_EnableProtectionUnit

Syntax

#include <nitro/os.h>
void OS_EnableProtectionUnit( void );

Arguments

None.

Return Values

None.

Description

Enables the protection unit.

Internal Operation

Modifies register 1 of the system control co-processor CP15.

See Also

OS_DisableProtectionUnit
