OS_IsRestrictPrepaidPoint

Syntax

#include <twl/os.h>

BOOL OS_IsRestrictPrepaidPoint(void);

Arguments

None.

Return Values

Accesses information regarding Parental Controls system settings and returns TRUE if restrictions have been placed on use of Nintendo DSi points.
Returns FALSE if Parental Controls settings have not been made and restrictions have not been placed on starting the Nintendo DSi browser.

Description

Accesses information regarding Parental Controls system settings and determines if restrictions have been placed on the use of Nintendo DSi points.
