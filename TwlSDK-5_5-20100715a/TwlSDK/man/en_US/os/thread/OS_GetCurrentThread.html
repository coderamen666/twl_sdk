OS_GetCurrentThread

Syntax

#include <nitro/os.h>
OSThread* OS_GetCurrentThread( void );

Arguments

None.

Return Values

The pointer to the current thread.

Description

Gets the current thread.

Do not call this from an interrupt handler. It returns an unknown value when called from an interrupt handler.

See Also

OS_InitThread
