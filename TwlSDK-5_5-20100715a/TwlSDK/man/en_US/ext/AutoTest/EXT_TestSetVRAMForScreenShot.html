EXT_TestSetVRAMForScreenShot

Syntax

#include <nitro/ext/scrntest.h>

void EXT_TestSetVRAMForScreenShot( GXVRamLCDC vram )

Arguments

vram VRAM used for screen capture

Return Values

None.

Description

This function specifies the VRAM that the auto test module will use for screen capture.

See Also

EXT_TestScreenShot, EXT_TestTickCounter, EXT_TestResetCounter
