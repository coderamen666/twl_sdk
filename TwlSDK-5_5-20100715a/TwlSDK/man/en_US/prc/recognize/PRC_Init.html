PRC_Init

Syntax

#include <nitro/prc.h>

void PRC_Init(void);

Arguments

None.

Return Values

None.

Description

Initializes the pattern recognition API. Call this before calling any other PRC_*.

See Also

PRC_InitPrototypeDB*, PRC_InitInputPattern*, PRC_GetRecognizedEntry*
