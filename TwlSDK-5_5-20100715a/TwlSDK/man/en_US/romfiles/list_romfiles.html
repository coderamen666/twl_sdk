List of Built-In Program Features

Wireless-Related

WMTestTool.srl A program for various wireless-related tests.
mb_child_[NITRO/TWL].srl A test program that can perform the same Single-Card Play as the release version IPL.
RelayStation.srl A test program that provides features equivalent to the WXC library's chance-encounter communications.
DSDownloadStation.srl [North America] DS Download Station is a test program that provides features equivalent to the WXC library's chance-encounter communications. For more information, see DS Trial Distribution-Related Materials for North America, published separately.

NAND Application-Related

TwlNmenu.srl A tool for importing NAND applications into the system memory.
NandFiler.srl A tool for browsing NAND applications' save data.

TCL-Related

PhotoChecker.srl Tool that checks whether a file that the TCL library is not involved with was written to the photo: region.
