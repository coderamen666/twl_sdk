AES_SetKey

Syntax

#include <twl/aes.h>

AESResult AES_SetKey(const AESKey* pKey);

Arguments

pKey Key to set.

Return Values

Returns AES_RESULT_SUCCESS if the key was set successfully. Any other return values indicate an error.

Description

Sets the key used for AES encryption (decryption).
