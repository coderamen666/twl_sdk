CARD_VerifyEeprom

Syntax

#include <nitro/card.h>

static inline BOOL CARD_VerifyEeprom(u32 dst, const void *src, u32 len);

Arguments

dst Comparison destination EEPROM address.
src Memory address of comparison source.
len Byte length of data to be compared.

Return Values

TRUE if the process succeeds and the data are the same. FALSE otherwise.

Description

This function compares the EEPROM device data to the specified memory. The function does not return control until it has finished processing. If the process succeeds and the data matches, the function returns TRUE. Otherwise, it returns FALSE. To get the value when processing fails, call the CARD_GetResultCode function. To use this function, you must first lock the access bus with the CARD_LockBackup function.

This function communicates with the ARM7 processor internally via PXI.
The process will not end if PXI interrupts are not enabled.
