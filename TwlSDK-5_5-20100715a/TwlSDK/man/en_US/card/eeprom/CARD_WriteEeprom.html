CARD_WriteEeprom

Syntax

#include <nitro/card.h>

BOOL CARD_WriteEeprom(u32 dst, const void *src, u32 len);

Arguments

dst Transfer destination EEPROM address.
src Transfer source memory address.
len Byte length of data to be written.

Return Values

TRUE if the process succeeds. FALSE otherwise.

Description

This function writes data to an EEPROM device. The function does not return control until it has finished processing. It returns TRUE if the process was successful or FALSE if it failed. To get the value when processing fails, call the CARD_GetResultCode function. To use this function, you must first lock the access bus with the CARD_LockBackup function.

This function communicates with the ARM7 processor internally via PXI.
The process will not end if PXI interrupts are not enabled.
