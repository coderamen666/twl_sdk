CARD_IsAvailable

Syntax

#include <nitro/card.h>

BOOL CARD_IsAvailable( void );

Arguments

None.

Return Values

Returns TRUE if the CARD module is initialized. Otherwise returns FALSE.

Description

Determines if the CARD_Init function has already been called and the CARD module initialized.
