VIB_IsCartridgeEnabled

Syntax

#include <nitro/vib.h>
BOOL VIB_IsCartridgeEnabled(void);

Arguments

None.

Return Values

Returns TRUE if a Rumble Pak is present; otherwise, returns FALSE.

Description

Determines whether or not a Rumble Pak is plugged in. Always returns FALSE if the Rumble Pak has been removed and plugged in even once.)
