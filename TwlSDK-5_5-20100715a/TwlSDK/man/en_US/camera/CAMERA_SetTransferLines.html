CAMERA_SetTransferLines

Syntax

#include <twl/camera.h>

void CAMERA_SetTransferLines(int lines);

Arguments

lines Line count (1–16)

Return Values

None.

Description

Sets the number of lines to transfer to the buffer at once. Configure the amount of data to save in the line buffer so that it does not exceed 1024 pixels (2 KB), which is the capacity of the line buffer.
You can set the number of lines to the value found using CAMERA_GET_MAX_LINES.

If there are any odd amounts at the end of a frame, a new DMA will be started synchronously with the start of a camera V-Blank and only the remaining amount will be transferred. Specifically, this is the case for odd amounts at the end when lines and the height specified by a CAMERA_I2CSize* function or the CAMERA_SetTrimmingParams function are not divisible.

This function must be called before the CAMERA_Start or CAMERA_StartCapture function is used to start capturing.

See Also

CAMERA_Start, CAMERA_StartCapture
