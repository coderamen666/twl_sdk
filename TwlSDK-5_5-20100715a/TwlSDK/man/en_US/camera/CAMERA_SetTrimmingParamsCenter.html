CAMERA_SetTrimmingParamsCenter

Syntax

#include <twl/camera.h>

void CAMERA_SetTrimmingParamsCenter(u16 destWidth, u16 destHeight, u16 srcWidth, u16 srcHeight);

Arguments

destWidth Width after trimming (in pixels).
destHeight Height after trimming (in pixels).
srcWidth Camera resolution width (in pixels).
srcHeight Camera resolution height (in pixels).

Return Values

None.

Description

Specifies the positions to use for trimming. Trimming includes the pixel with an x-coordinateof (srcHeight-destHeight)/2 and a y-coordinate of (srcWidth-destWidth)/2, but not the pixel with an x-coordinate of (srcHeight+destHeight)/2 and a y-coordinate of srcWidth+destWidth)/2. The camera resolution width and height specified as arguments should be the same as the size specified to the CAMERA_I2CSize* functions.

To use this function, trimming must first be enabled by the CAMERA_SetTrimming function.

The width (destWidth) of the trimming range specified in this function must be an even number.

The following behavior will result if the trimming range is not specified properly with this function.

  • This function will do nothing and return when the trimming range exceeds the camera resolution size (destWidth > srcWidth or destHeight > srcHeight).

See Also

CAMERA_SetTrimming, CAMERA_SetTrimmingParams, CAMERA_I2CSize
