CAMERA_SetBufferErrorCallback

Syntax

#include <twl/camera.h>

void CAMERA_SetMasterCallback(CAMERAIntrCallback callback);

Arguments

callback Callback function that is invoked during a buffer error interrupt.

Return Values

None.

Description

Sets the callback function that is invoked during the camera's buffer error interrupt.
