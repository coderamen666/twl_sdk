CAMERA_GET_MAX_LINES

Definition

#include <twl/camera.h>

#define CAMERA_GET_MAX_LINES(width) MATH_MIN((1024 / (width)), 16)

Description

Calculates the maximum value that can be set with the CAMERA_SetTransferLines function. If the CAMERA_SetTrimmingParams function was used to specify a width, that width will be specified as the argument value. Otherwise, the width of the camera size set by a CAMERA_I2CSize* function will be specified as the argument value.

See Also

CAMERA_SetTransferLines, CAMERA_I2CSize*
