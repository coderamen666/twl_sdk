NAND Application (NA) Function List

Archives

NA_LoadOtherTitleArchive Loads the specified NAND application's save data region to the file system.
NA_UnloadOtherTitleArchive Unloads a save data region that has been loaded from another NAND application, removing it from the file system.
NA_FormatTitleArchive Initializes the save data region.
NATitleArchive This is an enumerated type that specifies types of NAND application save data.

Internal Fonts

NA_InitSharedFont Prepares to use internal fonts.
NA_GetSharedFontTableSize Gets the size of management information.
NA_LoadSharedFontTable Loads management information.
NA_GetSharedFontSize Gets the size for an internal font.
NA_LoadSharedFont Loads an internal font into memory.
NASharedFontIndex An enumerated type that specifies the internal fonts.

Sub-Banners

NA_SaveSubBanner Sets the sub-banner.
NA_LoadSubBanner Loads the sub-banner.
NA_DeleteSubBanner Unsets the sub-banner.
NA_MakeSubBannerHeader Creates a sub-banner header.
NASubBanner The sub-banner structure.
