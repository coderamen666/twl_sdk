NATitleArchive

Syntax

#include <twl/na.h>

typedef enum NATitleArchive
{
    NA_TITLE_ARCHIVE_DATAPUB,
    NA_TITLE_ARCHIVE_DATAPRV,
    NA_TITLE_ARCHIVE_MAX
}
NATitleArchive;

Elements

NA_TITLE_ARCHIVE_DATAPUB Specifies public save data.
NA_TITLE_ARCHIVE_DATAPRV Specifies private save data.

Description

This is an enumerated type that specifies types of NAND application save data.
