TWL-SDK MB Library (Overview)

The MB library is used to distribute NTR programs to both TWL and NTR systems.

Programs Distributed Using Multiboot

Both NITRO ROMs and HYBRID ROMs can be distributed from a multiboot parent to a child. LIMITED ROMs, which only run on TWL systems, cannot be distributed.

Accordingly, download play that uses the MB library cannot use TWL-specific feature like the cameras or DSP.

Also, because LIMITED ROMs cannot be distributed as stated above, the parent ROM must be either a NITRO ROM or a HYBRID ROM when using the clone boot feature. Elect to use a HYBRID ROM if you want the parent to be able to use TWL-specific features like the camera.

Note

The MB library only distributes the NITRO ROM portion. The extended region of TWL-exclusive executable binaries contained within HYBRID ROMs will not be distributed; the program that is received by the child is equivalent to a NITRO ROM.
