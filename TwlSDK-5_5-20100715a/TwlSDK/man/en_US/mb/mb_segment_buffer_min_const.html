MB_SEGMENT_BUFFER_MIN

Definition

#include <nitro/mb.h>
#define MB_SEGMENT_BUFFER_MIN   0x10000


Description

This constant indicates the minimum allowable size of the buffer passed as an argument to the MB_ReadSegment function. For more information, see the MB_GetSegmentLength function.

See Also

MB_GetSegmentLength, MB_ReadSegment
