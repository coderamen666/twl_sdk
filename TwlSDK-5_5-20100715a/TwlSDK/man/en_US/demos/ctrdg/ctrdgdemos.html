Game Pak Sample Demo

Location

$TwlSDK/build/demos/ctrdg

Compilation

Move to the $TwlSDK/build/demos/ctrdg directory and enter the following:

% make

 The SRL executable file and the NEF or TEF files having binary information are created in $TwlSDK/build/demos/ctrdg/(sample name)/bin/(platform)/(target)/.

 For example:

$TwlSDK/build/demos/ctrdg/pullout-1/bin/ARM9-TS/Release/main.srl

will be the executable file for the pullout-1 demo.

Execution

For NITRO ROMs:

If using IS-NITRO-DEBUGGER:

  • Load and execute an NLF file with IS-NITRO-DEBUGGER.
  • Execute an SRL file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER using the loadrun.exe tool from a command line.

For TWL/NITRO HYBRID ROMs:

If using IS-NITRO-DEBUGGER:

  • Execute an SRL file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER using the loadrun.exe tool from a command line.

Description


Name Directories Operation
pullout-1 demos Detects removal of Game Pak and Game Card, displays warning message, and stops.
backup-1 demos Accesses the AGB backup device using the CTRDG library.

Operation

The CTRDG library demos will only work with IS-NITRO-DEBUGGER.
