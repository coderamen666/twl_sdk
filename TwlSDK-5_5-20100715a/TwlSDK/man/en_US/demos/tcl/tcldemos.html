TCL Application Sample Demos

Location

$TwlSDK/build/demos.TWL/tcl

Compilation

Move to the $TwlSDK/build/demos.TWL/tcl directory and enter the following.

% make

 Compilation starts, and a TAD file in NAND application format is created in

$TwlSDK/build/demos.TWL/tcl/(sample_name)/bin/(platform)/(target)/

   For example:

$TwlSDK/build/demos.TWL/tcl/tcl-1/bin/ARM9-TS.LTD/Release/tcl_1.tad

The above path shows the file used to import the simple demo.

Execution

Perform either of the following procedures.

  • Open a TAD file using the IS-TWL-DEBUGGER software, and then import and run it.
  • Use $TwlSDK/bin/ARM9-TS/Release/TwlNmenu.srl to import and then run a TAD file that is stored on an SD Card.

Description

Name Directory Operation
tcl-1 demos.TWL Sample for displaying or deleting photo data from the photo region.
tcl-2 demos.TWL Sample for adding photo data to the photo region.

Operation

These demos do not work properly with ensata.
