Rumble Pak Sample Demo

Location

$TwlSDK/build/demos/vib

Compilation

Move to the $TwlSDK/build/demos/vib directory and enter the following:

% make

 Compiling will begin, and the SRL executable file and the NEF or TEF files having binary information will be generated in $TwlSDK/build/demos/vib/(sample name)/bin/(platform)/(target)/.

 For example:

$TwlSDK/build/demos/vib/pulse_vib/bin/ARM9-TS/Release/main.srl

will be the executable file for the pulse_vib demo.

Execution

For NITRO ROMs

If using IS-NITRO-DEBUGGER:

  • Load and execute an NLF file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER using the loadrun.exe tool from a command line

For TWL/NITRO Hybrid ROMs

If using IS-NITRO-DEBUGGER:

  • Execute an SRL file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER using the loadrun.exe tool from a command line

Description

Name Directory Operation
pulse_vib demos Sample that allows any pulse to be generated for the Rumble Pak.

Operation

The pulse_vib demo works properly only with the IS-NITRO-DEBUGGER.
