WM Block Transfer Library (WBT) Sample Demos

Location

$TwlSDK/build/demos/wbt

Compilation

Move to the $TwlSDK/build/demos/wbt directory and enter

% make

 Compiling will begin, and the SRL executable file and the NEF or TEF files having binary information will be generated in $TwlSDK/build/demos/wbt/(sample name)/bin/(platform)/(target)/.

 For example:

$TwlSDK/build/demos/wbt/wbt-1/bin/ARM9-TS/Release/main.srl

will be the executable file for the wbt-1 demo.

Execution

For NITRO ROMs

If using IS-NITRO-DEBUGGER:

  • Load and execute an NLF file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER using the loadrun.exe tool from a command line

If using IS-TWL-DEBUGGER:

  • Load and execute an NLF file with IS-TWL-DEBUGGER
  • Execute an SRL file with IS-TWL-DEBUGGER
  • Execute an SRL file with IS-TWL-DEBUGGER using the loadrun.TWL.exe tool from a command line

For TWL/NITRO Hybrid ROMs

If using IS-NITRO-DEBUGGER:

  • Execute an SRL file with IS-NITRO-DEBUGGER
  • Execute an SRL file with IS-NITRO-DEBUGGER using the loadrun.exe tool from a command line

If using IS-TWL-DEBUGGER:

  • Load and execute a TLF file with IS-TWL-DEBUGGER
  • Execute an SRL file with IS-TWL-DEBUGGER
  • Execute an SRL file with IS-TWL-DEBUGGER using the loadrun.TWL.exe tool from a command line

For TWL Limited ROMs

If using IS-TWL-DEBUGGER:

  • Load and execute a TLF file with IS-TWL-DEBUGGER
  • Execute an SRL file with IS-TWL-DEBUGGER
  • Execute an SRL file with IS-TWL-DEBUGGER using the loadrun.TWL.exe tool from a command line

Description

Name Directory Operation
wbt-1 demos A simple sample demonstrating the basic functions of WBT library.

Operation

The WBT library demos do not work properly on ensata.
