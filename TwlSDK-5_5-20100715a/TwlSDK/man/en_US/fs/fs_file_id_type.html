FSFileID

Definition

#include <nitro/fs.h>

typedef struct
{
/* private: (except direct operation with FSArchive) */
struct FSArchive *arc; /* attached archive */
u32 file_id; /* index of FAT of attached archive */
} FSFileID;

Description

This expresses an ID of a unique file contained in the file system.
