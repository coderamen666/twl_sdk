FS_GetArchiveBase

Syntax

#include <nitro/fs.h>

static inline u32 FS_GetArchiveBase(const FSArchive *p_arc);

Arguments

p_arc Address of the FSArchive structure.

Return Values

The archive's base offset.

Description

Returns the base offset of the archive.
If the archive has not been loaded to the system by FS_LoadArchive, calling this function returns an undefined value.

See Also

FSArchive, FS_LoadArchive, FS_UnloadArchive
