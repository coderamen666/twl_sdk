FS_UnregisterEventHook

Syntax


#include <nitro/fs.h>

void FS_UnregisterEventHook(FSEventHook *hook);

Arguments

hook Pointer to the hook structure used during registration with FS_RegisterEventHook().

Return Values

None.

Description

Unregisters an archive-monitoring hook that was registered using FS_RegisterEventHook().
