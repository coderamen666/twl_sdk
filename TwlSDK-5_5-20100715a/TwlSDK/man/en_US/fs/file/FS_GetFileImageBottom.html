FS_GetFileImageBottom

Syntax

#include <nitro/fs.h>

static inline u32 FS_GetFileImageBottom(const FSFile *p_file);

Arguments

p_file Address of the FSFile structure
This file information must be maintained.

Return Values

Returns the end of the file image area's offset value in the archive.

Description

Returns the end of the file image area's offset value in the archive. The return value is undefined when this function is called by any files other than the one opened by the FS_OpenFileEx function.
