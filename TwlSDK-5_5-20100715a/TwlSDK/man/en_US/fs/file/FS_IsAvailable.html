FS_IsAvailable

Syntax

#include <nitro/fs.h>

BOOL FS_IsAvailable(void);

Arguments

None.

Return Values

If the ROM files system is initialized, it returns TRUE. Otherwise it returns FALSE.

Description

Determines if the FS_Init function has been called and the ROM file system has been initialized.

FS_Init
