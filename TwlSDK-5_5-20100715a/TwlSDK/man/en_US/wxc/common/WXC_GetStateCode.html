WXC_GetStateCode

Syntax

#include <nitro/wxc.h>

WXCStateCode WXC_GetStateCode(void);

Arguments

None.

Return Values

WXCStateCode enumerated type value that indicates current internal state.

Description

Gets the current internal status of the WXC library.
