WXC_GetUserBitmap

Syntax

#include <nitro/wxc.h>

u16 WXC_GetUserBitmap(void);

Arguments

None.

Return Values

Returns the status of currently connected users as a bitmap.

Description

Gets the status of currently connected users in the WXC library as a bitmap.
If communication has not been established, returns 0.
