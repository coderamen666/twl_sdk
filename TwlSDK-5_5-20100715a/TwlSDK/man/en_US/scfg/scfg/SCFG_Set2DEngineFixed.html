SCFG_Set2DEngineFixed

Syntax

#include <twl/scfg.h>
static inline void SCFG_Set2DEngineFixed( BOOL sw );

Arguments

sw If TRUE, revisions to the 2D engine circuit will be used.
If FALSE, the circuit will be identical to the NITRO-CPU's.

Return Values

None.

Description

Configures the use of revisions to the 2D engine circuit.

If revisions are specified to be used, BG window bugs will be fixed.

Revisions will not be used by default.

Note: Use this function when running on TWL hardware.

Because NITRO hardware does not include an SCFG block, this function is not expected to be called on NITRO hardware.
