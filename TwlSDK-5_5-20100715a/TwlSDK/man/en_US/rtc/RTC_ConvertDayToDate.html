RTC_ConvertDayToDate

Syntax

#include <nitro/rtc.h>
void RTC_ConvertDayToDate( RTCDate* date, s32 day );

Arguments

date Pointer to the buffer storing the date data.
day Total elapsed number of days, using January 1, 2000 as 0.

Return Values

None.

Description

Converts the total number of days that have elapsed since January 1, 2000 into an RTCDate type.
The day must be 0 or more and 36524 or lower. If less than 0 it is changed to 0 and if more than 36524 it is changed to 36524. Then the result is returned.

See Also

RTC_ConvertDateToDay, RTCDate
