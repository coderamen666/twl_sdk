DSP_WaitForG711

Syntax


void DSP_WaitForG711(void);

Arguments

None.

Return Values

None.

Description

Waits until the last G.711 function call has completed.
Use the DSP_TryWaitForG711 function to determine if it has completed.

Note

  • You must use the DSP_LoadG711 function to load the G.711 component before using this function.
