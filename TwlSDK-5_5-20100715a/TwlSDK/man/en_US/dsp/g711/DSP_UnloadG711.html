DSP_UnloadG711

Syntax


void DSP_UnloadG711(void);

Arguments

None.

Return Values

None.

Description

Unloads the G.711 component loaded by the DSP_LoadG711 function. WRAM-B and WRAM-C slots allocated while loading will be released internally and automatically by the library.

See Also

DSP_LoadG711 DSP_EncodeG711 DSP_DecodeG711 DSP_WaitForG711
