DSP_StopSound

Syntax


void DSP_StopSound(void);

Arguments

None.

Return Values

None.

Description

Stops any sound playback currently being performed by the DSP_PlaySound or DSP_PlayShutterSound function.

See Also

DSP_PlaySound DSP_PlayShutterSound DSP_IsSoundPlaying
