DSP_PlaySound

Syntax


void DSP_PlaySound(const void *src, u32 len, BOOL stereo);

Arguments

src The original sampling data.
Only 16-bit PCM data can be specified, and its sampling rate must match the I2S operating frequency obtained from the SNDEX_GetI2SFrequency function.
The specified buffer has the same restrictions as a 16-bit DMA transfer. See Note below for more details.
len Byte size of the sampling data.
stereo TRUE for stereo and FALSE for monaural.

Return Values

None.

Description

Plays an arbitrary sound from the DSP. If you call this function while playing another sound, the DSP stops the prior sound and plays the new sound. However, if you call it while DSP_PlayShutterSound is playing the shutter sound, it will be ignored. This function can be used if any of the DSP components have been loaded.

Note

  • The specified buffers must all satisfy the same conditions as 16-bit DMA transfers.
    • Addresses must be 16-bit aligned.
    • The buffer content must be cached in advance.
