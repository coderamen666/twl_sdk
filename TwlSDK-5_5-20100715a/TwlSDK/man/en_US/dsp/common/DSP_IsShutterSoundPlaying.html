DSP_IsShutterSoundPlaying 

Syntax


BOOL DSP_IsShutterSoundPlaying(void);

Arguments

None.

Return Values

Returns TRUE if the shutter sound is being played as the sound output from the DSP; FALSE otherwise.

Description

This function determines whether the shutter sound is being played as the sound output from the DSP using DSP_PlayShutterSound(). This function can be used if any of the DSP components have been loaded.
