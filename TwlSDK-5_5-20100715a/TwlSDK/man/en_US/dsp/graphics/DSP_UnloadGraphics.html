DSP_UnloadGraphics

Syntax


BOOL DSP_UnloadGraphics(void);

Arguments

None.

Return Values

None.

Description

Unloads the graphics component loaded by the DSP_LoadGraphics function. WRAM-B and WRAM-C slots allocated while loading will be released internally and automatically by the library.

See Also

DSP_LoadGraphics, DSP_ConvertYuvToRgb, DSP_ConvertYuvToRgbAsync,
DSP_Scaling, DSP_ScalingEx, DSP_ScalingAsync, DSP_ScalingAsyncEx
