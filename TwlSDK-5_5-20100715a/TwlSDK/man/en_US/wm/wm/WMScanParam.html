WMScanParam

Definition

#include <nitro/wm.h>

typedef struct WMScanParam
{
    WMBssDesc*  scanBuf;
    u16         channel;
    u16         maxChannelTime;
    u8          bssid[ 6 ];
    u16     rsv[9];
} WMScanParam;

Elements

scanBuf Pointer to the buffer that stores the information about the parent devices discovered as a result of the search. Be sure to watch the cache because ARM7 directly writes this data to the buffer.
channel Wireless channel to scan. Use a value in the range of 1-14.
maxChannelTime Maximum disconnect-scan time (in milliseconds). Set to a value from 10 through 1,000. Normally, the value obtained in the WM_GetDispersionScanPeriod function should be designated here.
bssid MAC address of the parent device that is the target of the search. If everything is 0xFF, there will be no extraction via the MAC address, and all parent devices will be the target of the search.

Description

This structure is a collection of the information that must be set on a child machine when it begins scanning for a parent.
