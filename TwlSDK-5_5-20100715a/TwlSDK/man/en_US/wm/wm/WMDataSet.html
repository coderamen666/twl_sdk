WMDataSet

Definition

#include <nitro/wm.h>

typedef struct WMDataSet
{
    u16 aidBitmap;
    u16 receivedBitmap;
    u16 data[WM_DS_DATA_SIZE/sizeof(u16)];
} WMDataSet;

Elements

aidBitmap AID bitmap of device performing data sharing. Determines the data structure of data.
receivedBitmap Bitmap of AID with valid data in data. For unconnected children, the bit is 0.
data Shared data array. The shared data of aidBitmap-enabled device is packed in order from low AID of each shared size. The shared size is not held in WMDataSet, so you need to get it separately when dividing.

Description

The structure showing the shared data obtained with the data sharing function of WM. In general, access this structure with the WM_GetSharedDataAddress function.
