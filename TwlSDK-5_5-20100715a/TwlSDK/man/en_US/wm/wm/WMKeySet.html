WMKeySet

Definition

#include <nitro/wm.h>

typedef struct WMKeySet
{
    u16 seqNum;
    u16         rsv;
    u16 key[16];
} WMKeySet;

Elements

seqNum Sequential number of the key set. After key sharing communication has been successful between connected machines, this number will increment by 1 each time key information is shared.
key Array of shared key input information, including its own. Data that has an index value of 0 is parent key information. Data with indexes 1–15 are each AID's child key information.

Description

This structure indicates shared key information that is obtained by the WM key sharing function.
