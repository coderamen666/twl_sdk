Elements

SNDEX_IIR_FILTER_ADC_1 ... 5 IIR filter for microphone input data (stage 1 to stage 5).
SNDEX_IIR_FILTER_RESERVED_1 ... 15 Reserved by the system for future use. Do not specify it.

Description

Enumerated type that indicates the target to which IIR filters will be applied.

Filters are applied in ascending numerical order, as follows.

  • (Microphone) -> A/D converter → ADC1 → ADC2 → ADC3 → ADC4 → ADC5 → microphone input data

SNDEX_IIR_FILTER_RESERVED_nn (nn=1～15) is reserved by the system, and must not be specified by users.
