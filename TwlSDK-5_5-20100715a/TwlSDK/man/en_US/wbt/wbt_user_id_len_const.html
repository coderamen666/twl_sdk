WBT_USER_ID_LEN

Definition

#include <nitro/wbt.h>

#define WBT_USER_ID_LEN 32

Description

This is the maximum size of the user-defined information that can be assigned to blocks registered using the WBT_RegisterBlock function. The other parties can get this detailed information on their side with the WBT_GetBlockInfo function.
