WBTRequestSyncCallback

Definition

#include <nitro/wbt.h>

typedef	struct
{
	WBTBlockNumEntry num_of_list;
	s16 peer_packet_size;
	s16 my_packet_size;
	u16 pad1;
	u32 padd2[2];
} WBTRequestSyncCallback;

Description

The structure passed by the completion callback for the WBT_RequestSync function.

The details of each member are as follows.
num_of_list Total number of data blocks registered on the other side.
peer_packet_size MP communication packet size sent from the other side.
my_packet_size MP communication packet size sent to the other side.
