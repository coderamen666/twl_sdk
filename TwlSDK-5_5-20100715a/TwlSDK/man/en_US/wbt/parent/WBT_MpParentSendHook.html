WBT_MpParentSendHook 

Syntax

#include <nitro/wbt.h>

int WBT_MpParentSendHook(void *sendbuf, int send_size);

Arguments

sendbuf Parent's send buffer, used for WBT by functions such as WM_SetMPDataToPort during MP communications.
send_size sendbuf parent packet size allocated for WBT. This value must be larger than WBT_PACKET_SIZE_MIN.

Return Values

Returns the byte size stored by the actual data to send.

Description

Stores in the send buffer the data the WBT library parent sends by MP communication. Data based on the WBT library protocol is set in the send buffer. Various commands are executed by repeating this function during the MP transmission. Note: This function merely stores data in the buffer. After this, send the buffer using a function such as WM_SetMPDataToPort.)

When data receipt is requested by WBT_GetBlock from any communication destination, a WBT_CMD_PREPARE_SEND_DATA callback may be generated in this call, depending on settings.
