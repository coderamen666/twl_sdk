tad2rom

Description

The tad2rom tool generates an SRL file (.tad.srl) that includes the specified TAD file.

The same processing represented by maketad with the -n option is performed on the TAD file.

Using the Tool

Start Command

% tad2rom TADFILE

TADFILE represents the TAD file to be included in the SRL file to be output. A file named TADFILE.srl is output.

To import data from the included TAD file, insert a DS card on which the output SRL has been written into the DS card slot of the TWL unit and start the TwlNmenu imported into system save memory.

The -h option displays simple instructions.

Location

$TwlSDK/tools/bin/tad2rom.exe
