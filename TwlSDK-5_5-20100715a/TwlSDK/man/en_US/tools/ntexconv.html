ntexconv

Description

ntexconv is a tool for converting BMP or TGA format image files into the format that is used by TWL libraries, etc.

Using the Tool

For information on using ntexconv, see the ntexconv manual (PDF).

Location

$TwlSDK/tools/bin/ntexconv.exe
