MI_GetNDmaIntervalTimer

Syntax

#include <twl/mi.h>

u32 MI_GetNDmaIntervalTimer( u32 ndmaNo );

Arguments

ndmaNo DMA number to get settings for (0-3).

Return Values

Timer value of the block transfer interval timer.

Description

Gets the value of the block transfer interval timer setting.

It gets the value of the block transfer interval timer setting for the new TWL DMA specified in ndmaNo.
