MI_StopAllDma

Syntax

#include <nitro/mi.h>

void MI_StopAllDma( void )

Arguments

None.

Return Values

None.

Description

Stops all DMA channels. (However, all DMA channels does not include the TWL's new DMA channels.)

This function calls the MI_StopDma function for DMA channels 0-3.
