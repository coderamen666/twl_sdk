EL_GetResultCode

Syntax

#include <twl/el.h>

ELResult EL_GetResultCode( void);

Arguments

None.

Return Values

Returns the result of the last process performed by the EL library.

Description

Returns the result of the last process performed by the EL library.
If this function has been called before the EL_Init function runs, it returns EL_RESULT_FAILURE.
