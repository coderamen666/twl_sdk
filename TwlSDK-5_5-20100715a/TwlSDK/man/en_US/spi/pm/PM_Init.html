PM_Init

Syntax

#include <nitro/spi.h>
void PM_Init( void );

Arguments

None.

Return Values

None.

Description

Initializes the PMIC manager.

The PM_Init function must be called before any other PM function; however, this is normally not necessary because it is called from OS_Init.

See Also

None.
