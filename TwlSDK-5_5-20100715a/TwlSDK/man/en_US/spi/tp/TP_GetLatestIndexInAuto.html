TP_GetLatestIndexInAuto

Syntax

#include <nitro/spi.h>
u16 TP_GetLatestIndexInAuto( void );

Arguments

None.

Return Values

None.

Description

This function acquires the index of the last data that autosampling stored in the loop buffer.
