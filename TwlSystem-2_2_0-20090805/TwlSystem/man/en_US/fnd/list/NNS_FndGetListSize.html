NNS_FndGetListSizse

Syntax

#include <nnsys/fnd.h>
u16 NNS_FndGetListSize( NNSFndList* list );

Arguments

list Pointer to the list structure.

Return Values

Returns the number of objects registered to the list structure.

Description

Gets the number of objects registered to the list structure.
