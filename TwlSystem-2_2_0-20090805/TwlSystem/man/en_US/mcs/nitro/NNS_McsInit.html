NNS_McsInit

Syntax

#include <nnsys/mcs.h>
void NNS_McsInit(void* workMem);

Arguments

workMem Work memory used by MCS. Specify a memory region of NNS_MCS_WORKMEM_SIZE bytes that is aligned to 4.

Return Values

None.

Description

Performs initialization for communication with devices. It must be called before any other NNS_Mcs functions are used.

See Also
