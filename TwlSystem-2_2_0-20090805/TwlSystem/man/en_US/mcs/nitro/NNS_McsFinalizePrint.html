NNS_McsFinalizePrint

Syntax

#include <nnsys/mcs.h>
void NNS_McsFinalizePrint();

Arguments

None.

Return Values

None.

Description

Performs end processing for the string output functions.

Frees work memory specified by the NNS_McsInitPrint function.

Call this function before calling the NNS_McsFinalize function.

See Also

NNS_McsInitPrint
