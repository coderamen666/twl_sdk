NNS_SndArcGetLoadBlockSize

Syntax

#include <nnsys/snd.h>
s32 NNS_SndArcGetLoadBlockSize( void );

Arguments

None.

Return Values

Returns the load block size.

Description

Gets the load block size when data is loaded. This size is specified in the current sound archive.

See Also

NNS_SndArcSetLoadBlockSize
