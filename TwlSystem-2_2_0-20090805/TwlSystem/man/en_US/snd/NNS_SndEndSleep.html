NNS_SndEndSleep

Syntax

#include <nnsys/snd.h>
void NNS_SndEndSleep( void );

Arguments

None.

Return Values

None.

Description

It is no longer necessary to call this function at sleep time. When this function is called, it no longer does anything.

See Also

None.

Revision History

2004/10/07 Deleted because it is no longer necessary.
2004/09/02 Added text about prohibiting Sound playback for 15 milliseconds after awakening from sleep.
