NNS_SndBeginSleep

Syntax

#include <nnsys/snd.h>
void NNS_SndBeginSleep( void );

Arguments

None.

Return Values

None.

Description

It is no longer necessary to call this function at sleep time. When this function is called, it no longer does anything.

See Also

None.
