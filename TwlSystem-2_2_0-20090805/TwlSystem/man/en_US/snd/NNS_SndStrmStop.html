NNS_SndStrmStop

Syntax

#include <nnsys/snd.h>
void NNS_SndStrmStop( NNSSndStrm* stream );

Arguments

stream Pointer to a stream structure.

Return Values

None.

Description

Stops stream playback.

Note

To resume stream playback, the NNS_SndStrmSetup function needs to be called again.

See Also

NNS_SndStrmSetup
