NNSG3dResTex

Description

The NNSG3dResTex structure corresponds to a collection of textures and palettes. This structure serves as the access point to data relating to G3D textures and palettes. The NNSG3dResTex structure and the texture and palette data exist as data in the .nsbmd and .nsbtx files. The data are created when the .imd file is converted by g3dcvtr.

The following data can be accessed via the NNSG3dResTex structure:

  • Keys to the storage location for normal texture, 4x4 texel compressed texture, and palette data
  • Information on each texture and palette size and other characteristics
  • Image of each texture and palette

This structure and the keys and other information obtained through this structure are not changed by the G3D library during rendering (except when changed from within a callback).
