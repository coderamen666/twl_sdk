NNS_G3dReleaseMdlSet

Syntax

#include <nnsys/g3d/kernel.h>
void
NNS_G3dReleaseMdlSet(
    NNSG3dResMdlSet* pMdlSet
);

Arguments

pMdlSet The pointer to the model set

Return Values

None.

Description

Releases the bindings to texture and palette for every model in the set pMdlSet.

See Also

NNS_G3dBindMdlSet
