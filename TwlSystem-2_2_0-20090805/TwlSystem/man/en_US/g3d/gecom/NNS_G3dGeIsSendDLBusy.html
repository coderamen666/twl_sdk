NNS_G3dGeIsSendDLBusy

Syntax

#include <nnsys/g3d/gecom.h>
BOOL NNS_G3dGeIsSendDLBusy(void);

Arguments

None.

Return Values

Returns TRUE during DMA transfer of geometry commands. Otherwise returns FALSE.

Description

Returns TRUE when DMA transfer of geometry commands via the NNS_G3dGeSendDL function is in the process of being carried out.

See Also

NNS_G3dGeWaitSendDL, NNS_G3dGeSendDL
