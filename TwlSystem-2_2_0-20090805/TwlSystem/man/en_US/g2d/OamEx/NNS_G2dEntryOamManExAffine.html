NNS_G2dEntryOamManExAffine

Syntax

#include <nnsys/g2d/g2d_OAMEx.h>
u16  NNS_G2dEntryOamManExAffine( 
                                NNSG2dOamManagerInstanceEx*     pMan, 
                                const MtxFx22*                  mtx );

Arguments

pMan [OUT] Extended OAM manager entity
mtx [IN] Affine matrix

Return Values

The registered affine index

Description

Registers an affine index in the extended OAM manager. The name of this function has been changed from NNS_G2dEntryNewAffineParamsEx. The previous function remains under a different name in order to maintain compatibility.

See Also

NNS_G2dEntryNewAffineParamsEx
