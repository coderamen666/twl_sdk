NNS_G2dResetMCCellAnimationAll

Syntax

#include <nnsys/g2d/g2d_MultiCellAnimation.h>

void NNS_G2dResetMCCellAnimationAll
(
    NNSG2dMultiCellInstance*    pMCInst 
);

Arguments

pMCInst [OUT] Multicell entity

Return Values

None.

Description

Resets the animation frames for the cell animations that compose the multicell entity.
