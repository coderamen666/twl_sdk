NNS_G2dEndRendering

Syntax

#include <nnsys/g2d/g2d_Renderer.h>
void NNS_G2dEndRendering();

Arguments

None.

Return Values

None.

Description

Call after the renderer drawing. This function must be called after the renderer drawing method (NNS_G2dDrawXXX) is called. Do not call this function inside the BeginRendering — EndRendering block. (Nested function calls are prohibited.)

Restores the internal state, that was changed after rendering, to its initial state.
If the optimization hint flag was set (NNS_G2dBeginRenderingEx is used), this function performs optimization post-processing.
The optimization hint flag is reset by this function. In other words, the optimization hint flag is valid only within the renderer's BeginRendering — EndRendering block.
