NNS_G2dScale

Syntax

#include <nnsys/g2d/g2d_Renderer.h>
void NNS_G2dScale(fx32 x, fx32 y, fx32 z );

Arguments

x [IN] Scale x
y [IN] Scale y
z [IN] Scale z

Return Values

None.

Description

Multiplies the current matrix of the renderer's internal matrix stack by the scale matrix.

See Also

NNS_G2dTranslate, NNS_G2dRotZ
