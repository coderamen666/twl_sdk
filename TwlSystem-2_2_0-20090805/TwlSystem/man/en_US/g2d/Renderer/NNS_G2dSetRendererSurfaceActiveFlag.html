NNS_G2dSetRendererSurfaceActiveFlag

Syntax

#include <nnsys/g2d/g2d_Renderer.h>

	NNS_G2D_INLINE void NNS_G2dSetRendererSurfaceActiveFlag
	( 
		NNSG2dRenderSurface*    pSurface, 
		BOOL                    bActive 
	);

Arguments

pSurface [OUT] Renderer surface entity
bActive [IN] Whether active or inactive. TRUE for active.

Return Values

None.

Description

Sets the active state of the render surface.

See Also

NNS_G2dGetRendererSurfaceActiveFlag
