NNS_G2dIsImagePaletteReadyToUse

Syntax

#include <nnsys/g2d/g2d_Image.h>
BOOL NNS_G2dIsImagePaletteReadyToUse( 
                        const NNSG2dImagePaletteProxy*     pImg, 
                        NNS_G2D_VRAM_TYPE                  type );

Arguments

pImg [IN] Image palette proxy
type [IN] VRAM type to use (3D, 2D-Main, or 2D-Sub)

Return Values

TRUE if useable (already configured).

Description

Checks if the image palette proxy has been properly configured for the type of VRAM usage specified in type, and is thus in a usable state. You can specify any of the following types of VRAM usage. 

    NNS_G2D_VRAM_TYPE_3DMAIN    This type is used for the 3D graphics engine
    NNS_G2D_VRAM_TYPE_2DMAIN    This type is used for the 2D graphics engine A
    NNS_G2D_VRAM_TYPE_2DSUB     This type is used for the 2D graphics engine B

See Also

None.
