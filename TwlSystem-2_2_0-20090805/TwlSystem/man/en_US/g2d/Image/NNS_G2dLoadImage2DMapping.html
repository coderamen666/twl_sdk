NNS_G2dLoadImage2DMapping

Syntax

#include <nnsys/g2d/g2d_Image.h>
void NNS_G2dLoadImage2DMapping( 
                        const NNSG2dCharacterData*  pSrcData, 
                        u32                         baseAddr, 
                        NNS_G2D_VRAM_TYPE           type, 
                        NNSG2dImageProxy*           pImgProxy );

Arguments

pSrcData [IN] Input image data source
baseAddr [IN] Load destination address
type [IN] VRAM type to use (3D, 2D-Main, or 2D-Sub)
pImgProxy [OUT] Image proxy that stores the load result

Return Values

None.

Description

Loads the image data to the hardware and sets the load results and the image format to the image proxy.
This function is used exclusively for the 2D mapping mode's raw images.
If 2D Graphics Engine VRAM has been specified, this function internally sets the 2D Graphics Engine character mode to 2D mapping mode.

See Also

None.
