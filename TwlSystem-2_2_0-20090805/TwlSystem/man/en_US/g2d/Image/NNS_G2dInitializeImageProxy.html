NNS_G2dInitializeImageProxy

Syntax

#include <nnsys/g2d/g2d_Image.h>
void NNS_G2dInitializeImageProxy   ( NNSG2dImageProxy* pImg );

Arguments

pImg [OUT] Image proxy

Return Values

None.

Description

Initializes the image proxy. This function must always be called before using the image proxy.

The name of this function has changed to NNS_G2dInitImageProxy(). This function will remain as a separate name to maintain compatibility.
