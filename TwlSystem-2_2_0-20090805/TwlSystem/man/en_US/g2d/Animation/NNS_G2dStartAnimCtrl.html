NNS_G2dStartAnimCtrl

Syntax

#include <nnsys/g2d/g2d_Animation.h>

  static void NNS_G2dStartAnimCtrl
                            ( 
                                NNSG2dAnimController*       pAnimCtrl 
                            );

Arguments

pAnimCtrl [OUT] NNSG2dAnimController entity

Return Values

None.

Description

Starts animation playback.

Note: If a STOP-method animation sequence plays to the end, the animation controller is set to the Stop state.
Because of this, you will need to call the NNS_G2dStartAnimCtrl function for the animation controller to update the animation again.

This function is a renamed version of the NNS_G2dStartAnimController function. The past function remains as a separate name to maintain compatibility.
