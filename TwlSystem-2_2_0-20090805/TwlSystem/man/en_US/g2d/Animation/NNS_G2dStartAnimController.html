NNS_G2dStartAnimController

Syntax

#include <nnsys/g2d/g2d_Animation.h>

  static void NNS_G2dStartAnimController
                            ( 
                                NNSG2dAnimController*       pAnimCtrl 
                            );

Arguments

pAnimCtrl [OUT] NNSG2dAnimController entity

Return Values

None.

Description

Starts animation playback.
The name of this function has changed to NNS_G2dStartAnimCtrl. The past function remains as a separate name to maintain compatibility.
