NNS_G2dInitAnimController

Syntax

#include <nnsys/g2d/g2d_Animation.h>
void NNS_G2dInitAnimController( 
                      NNSG2dAnimController*     pAnimCtrl );

Arguments

pAnimCtrl [OUT] NNSG2dAnimController entity

Return Values

None.

Description

Initializes the NNSG2dAnimController entity. Be absolutely sure to call this before using the NNSG2dAnimController entity.

The name of this function has changed to NNS_G2dInitAnimCtrl. The past function remains as a separate name to maintain compatibility.
