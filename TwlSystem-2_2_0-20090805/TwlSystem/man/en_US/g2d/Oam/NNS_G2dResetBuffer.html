NNS_G2dResetBuffer

Syntax

#include <nnsys/g2d/g2d_Oam.h>
void NNS_G2dResetBuffer( NNSG2dOamManagerInstance* pMan );

Arguments

pMan [OUT] OAM manager entity.

Return Values

None.

Description

Resets the internal OAM buffer for the range that the OAM manager is managing. (It is forcibly overwritten with the default value that sets the OBJ outside the screen.)
The name of this function has changed to NNS_G2dResetOamManagerBuffer. To maintain compatibility with past versions, this function has been kept under a different name.
