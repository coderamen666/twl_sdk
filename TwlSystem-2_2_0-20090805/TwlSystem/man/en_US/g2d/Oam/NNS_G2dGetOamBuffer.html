NNS_G2dGetOamBuffer

Syntax

#include <nnsys/g2d/g2d_Oam.h>
GXOamAttr* NNS_G2dGetOamBuffer( NNSG2dOamType type );

Arguments

type [OUT] OAM buffer type

Return Values

Pointer to the OAM Manager's internal buffer

Description

Gets a pointer to the data buffer held by the OAM Manager within the module.

See Also

None.
