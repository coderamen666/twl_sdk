NNS_G2dGetOamAttrCapacity

Syntax

#include <nnsys/g2d/g2d_Oam.h>
u16 NNS_G2dGetOamAttrCapacity( NNSG2dOamManagerInstance* pMan );

Arguments

pMan [IN] OAM manager entity.

Return Values

The number of usable OBJs.

Description

Gets the number of OBJs that can be used by the OAM Manager. The name of this function has changed to NNS_G2dGetOamManagerOamCapacity(). To maintain compatibility with past versions, this function has been kept under a different name.
