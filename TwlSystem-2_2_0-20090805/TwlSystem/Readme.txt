1. Introduction
==============================================================================

  This package is for the Nintendo TWL-System.

  The Nintendo TWL-System is the collective name for the tools and libraries used to develop graphics and sound for Nintendo DS series game software.


2. Documentation
==============================================================================

    Basic information for use of the package is described in $TwlSystem/docs/ReleaseNotes.html, which is bundled with the package. Read it first.
  
    See $TwlSystem/docs/QuickStart.pdf for details on the procedures for setting up the Twl-System and building the TwlSystem library and its demos.
