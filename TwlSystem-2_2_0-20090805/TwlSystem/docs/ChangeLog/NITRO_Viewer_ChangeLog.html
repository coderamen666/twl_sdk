Applied changes made to TWL-System library (2.0.0 RC2).

Note 79-V02: (NITRO-Viewer) Changed the shortcut for launching Viewer

When the path to the TWLSYSTEM_ROOT environment variable was described using a slash as a delimiter, the shortcut for launching Viewer did not work correctly. This bug was fixed by changing the Viewer launch shortcut into a batch file.

Note 79-V03: (NITRO-Viewer) Support for Ensata

Added support for operation with NITRO Ensata.

Changes in 2.0.0 RC

Note 78-V01: (NITRO-Viewer) Updated for the TWL-System library (2.0.0 RC)

Applied changes made to TWL-System library (2.0.0 RC).

Note 78-V02: (NITRO-Viewer) Changed environment variables configuration script

The script for setting environment variables (setViewerRootEnv.wsf/removeViewerRootEnv.wsf) has been revised so that it now sets TWLSYSTEM_ROOT. The environment variable that needed to be set in earlier versions, NITROVIEWER_ROOT, is now automatically set internally when NITRO-Viewer is launched.

Changes in Version 2007/11/26

Note 71-V01: (NITRO-Viewer) Updated for NITRO-System library (2007/11/26 version)

Applied changes made to NITRO-System Library (2007/11/26).

Note 71-V02: (NITRO-Viewer) Fixed bug where number of elements with the 3D preview is over 64

In 3D preview, a bug resulted in flashing for selections in 3D material editors when there were 64 or more elements for nodes, materials, and polygons. This has been fixed 

There is no official support, however, for displaying or editing models with 64 or more elements. Such models may not work normally.

Changes in Version 2005/07/22

Note 49-V01: (NITRO-Viewer) Added NITRO-Viewer functionality

Added automatic individual frame movement in a designated direction for each of the 2D operation targets.

Note 49-V02: (NITRO-Viewer) Changed rotation and scaling for 2D previews

Made adjustments so that double-size will always be enabled for objects in the 2D Player when scale/rotate is enabled but double-size is not. However, this adjustment is made only when performing scaling with the controller.

Changes in Version 2005/06/20

Note 47-V01: (NITRO-Viewer) Updated for VRAM content display functionality in 3D material editors

Added support for VRAM content display in the 3D Material Editor.

Note 47-V02: (NITRO-Viewer) Added resident model display functionality

Added resident module display functionality.

Note 47-V03: (NITRO-Viewer) Revised icon display when camera moves in 3D preview

Revised specifications so that icons displayed for rotational movement and translation modes are swapped when the camera is moved during a 3D preview.

Changes in Version 2005/06/06

Note 45-V01: (NITRO-Viewer) Updated for NITRO-System library (2005/06/06 version)

Applied changes made to NITRO-System Library (2005/06/06).

Note 45-V02: (NITRO-Viewer) Set waiting time for until start of key repeat

Created a wait period before key repetition starts so that moves in units of one dot can be made easily when previewing from NITRO-CHARACTER.

Note 45-V03: (NITRO-Viewer) Fixed bug where 3D side camera also moved during 2D operations

When previewing in 2D and 3D simultaneously in shared mode, the camera on the 3D side moved during 2D operation. This has been fixed.

Note 45-V04: (NITRO-Viewer) Fixed bug where background moved dramatically

Under certain conditions, when previewing a rotated and scaled background, it would move considerably. This has been fixed.

Note 45-V05: (NITRO-Viewer) Fixed bug where object for which rotate/scale is enabled is displayed without rotate/scale

When an object's rotation and scaling was enabled, and it was displayed without rotation or scaling, its display would malfunction. This bug has been fixed.

Changes in Version 2005/04/04

Note 44-V01: (NITRO-Viewer) Revised display of title screen when used with UIC

When using UIC, the title screen is now displayed on the screen before a connection is made from the NITRO-Viewer controller.

Note 44-V02: (NITRO-Viewer) Fixed bug where coordinate settings for windows in 2D preview do not work normally

When using window functionality in a 2D preview with NITRO-Viewer, the window coordinate settings did not work properly. This bug was fixed.

Note 44-V03: (NITRO-Viewer) Fixed bug where previews using background rotation or scaling do not work normally

When using background rotation or scaling in a 2D preview with NITRO-Viewer, malfunctions occurred in some cases. This bug was fixed.

Added support for the screen capture feature of 3D Material Editor.

Note 41-V02: (NITRO-Viewer) Added 3D animation functionality for Nintendo DS system controller

Added a feature that operates 3D animations with the Nintendo DS controller.

Changes in Version 2004/12/06

Note 34-V01: (NITRO-Viewer controller) Fixed bug in reconnecting with UIC

Corrected the problem that prevented 3D Material Editor or NITRO-CHARACTER from reconnecting when the connection was lost while switching between shared and exclusive modes or while changing VRAM settings while connected to NITRO-Viewer and UIC.

Note 34-V02: (NITRO-Viewer controller) Fixed bug where error is displayed when dialog box is closed

Under certain operating environments, an error would appear when closing the VRAM Bank Assignment or Version Information dialog boxes. This bug has been fixed.

Changes in Version 2004/11/15

Note 32-V01: (NITRO-Viewer) Added support for linear interpolation of fractional frames in ICA animation data

Added support for linear interpolation of fractional frames in ICA animation data.

Note 32-V02: (NITRO-Viewer) Support for 30 frame playback in 3D animation

Added support for 30 frame playback of 3D animation.

Note 32-V03: (NITRO-Viewer) Updated with option that stops after one playback of 3D animation

Added support for the option to stop the 3D animation after playing it once.

Note 32-V04: (NITRO-Viewer) Revised to enable selection and playback of specific animations

Added support to enable and disable the setting of multiple animations set for 3D preview and made it possible to select a specific animation.

Note 32-V05: (NITRO-Viewer) Updated for movement mode for three types of light

Added support for three types of light movement modes. In addition to the individual mode, lighting can move with the camera or be applied from the same direction as the camera.

Changes in Version 2004/10/19

Note 28-V01: (NITRO-Viewer) Fixed bug where interference was displayed in 2D preview

Fixed a bug that caused interference to be displayed when previewing a screen with a hidden background that was previously viewed using NITRO-Viewer 2D Preview.

Changes in Version 2004/10/12

Note 26-V01: (NITRO-Viewer) Support for previews using IS-NITRO-UIC

Added support for previews using IS-NITRO-UIC

Note 26-V02: (NITRO-Viewer controller) Support for communication with IS-NITRO-UIC

Added support for communication with IS-NITRO-UIC

Note 26-V03: Changed directory configuration

Created a controller directory and moved NITRO-Viewer Controller files to this directory. The NITRO-Viewer Controller DLL files nnsmcs.dll and nnsnvc.dll are stored in the lib directory because they are also used by applications that perform communication with the NITRO-Viewer Controller.

Note 26-V04: (NITRO-Viewer) Fixed memory leaks

Fixed a bug that caused a memory leak when loading 3D model data.

Note 26-V05: (NITRO-Viewer controller) Revised process resulting in communication error

Fixed a bug such that the NITRO-Viewer Controller does not freeze when a communication error occurs.

Changes in Version 2004/09/16

Note 21-V01: (NITRO-Viewer) Fixed bug that resulted in slower rendering during cell animation previews

Fixed a bug that caused rendering to be slow when previewing cell animations from NITRO-CHARACTER.

Note 21-V02: (NITRO-Viewer controller) Updated for previewing in Ensata

Supported preview on the Nintendo DS software emulator Ensata.

Note 21-V03: Added files

Added the NITRO-Viewer launch shortcut file (nsata_NITRO-Viewer_Controller) for Ensata.

Note 21-V04: (NITRO-Viewer controller) Fixed bug where error dialog box appears when shared mode settings are repeated

Fixed bug where error dialog box appears when shared mode settings are repeated.

Changes in Version 2004/09/02

Note 20-V01: (NITRO-Viewer) Added ability to choose preview image display screen

The preview image can be shown on either the upper or lower Nintendo DS screen.

Note 20-V02: (NITRO-Viewer) Updated for previewing from NITRO-CHARACTER

The viewer now supports preview from NITRO-CHARACTER (with some restrictions).

Note 20-V03: (NITRO-Viewer Controller) Language support

NITRO-Viewer controller now supports English and Japanese.

Note 20-V04: Changes to G2D binary converter used

The G2D binary converter has been changed from G2DConv.exe to g2dcvtr.exe.

Note 20-V05: Changed message display language for default settings script

The message that the initial setting script displays is now in English.

Note 20-V06: (NITRO-Viewer) Revised spelling errors in dialog boxes

The spelling on the Cancel button has been revised in the Assign VRAM Banks dialog box.

Changes in Version 2004/08/17

Note 17-V01: (NITRO-Viewer) Updated to preview IMA and ITA files

The viewer supports the preview of 3D material color animation (IMA) and texture SRT animation (ITA) files.

Note 17-V02: (NITRO-Viewer) Added process meter and animation bar indicating rendering time

A process meter has been added to indicate processing time for rendering 3D models and 3D animation and an animation bar to indicate the currently displayed frame of a 3D animation.

Note 17-V03: (NITRO-Viewer controller) Changed version display for ISNITRO.dll

The ISNITRO.dll version display was changed from 3 to 4 digits.

Note 17-V04: Added DLL file for NITRO-Viewer controller

A lib directory has been added to the NITRO-Viewer package. The directory contains two additional DLLs (dynamic-link libraries) for the NITRO-Viewer controller.
