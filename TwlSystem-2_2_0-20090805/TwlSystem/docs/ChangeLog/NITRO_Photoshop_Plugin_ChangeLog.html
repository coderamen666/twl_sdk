Note 77-H01: Ended support for Photoshop 7.0.1

Photoshop 7.0.1 is no longer supported.

Note 77-H02: Corrected a problem occurring when the Windows screen DPI setting was larger than 96 DPI

When the Windows screen DPI was set larger than 96 DPI, the dialog boxes did not display in their entirety. That has been corrected.

Changes in Version 2007/09/10

Note 69-H01: Added support for Windows version of Adobe Photoshop CS3

Added support for the Windows version of Adobe Photoshop CS3.

Note 69-H02: Corrected the method for saving the palette name

The save process was revised so that, if any spaces exist before or after the name specified using the Palette Name option, the spaces are deleted before saving.

Note 47-H04: Corrected problem where colors in some areas became darker when creating a 4x4 texel compressed texture

Corrected a problem where colors in some areas became darker if Use Linear Interpolation was OFF when creating a 4x4 texel compressed texture. When palette data is saved in 4x4 texel compressed texture format or when a 4x4 texel compressed texture filter is run on palette data, sometimes the amount of palette data increases compared to data from previous versions of the plug-in.
