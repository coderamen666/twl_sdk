Changes in 2.1.0

Note 1-D01: Makefile support for version.h

Included the $TwlSystem/include/nnsys/version.h file from Makefile and enabled the acquisition of version information.

Note 81-D02: Deleted the IDE sample demo

The IDE sample demo and IDE build guide have been deleted from the TWL-System package because they will be released in a different package.

Changes in 2.0.0 RC

Note 78-D01: Changed name to TWL-System

Changed NITRO-System to TWL-System.

In line with this change, the name of the environment variable needed for setting the root folder has been changed to TWLSYSTEM_ROOT from NITROSYSTEM_ROOT.

Note 78-D02: Changed the way versions are managed

Previously, the package version was managed using dates. That has been changed so now it is managed using version numbers.

Note that in line with this change, the version macro defined by $TwlSystem/include/nnsys/version.h has also been changed.

Changes in Version 2008/04/08

Note 76-D01: Support for TWL-SDK

Added support for TWL-SDK. The SDK that is supported is TWL-SDK 5.0 FC plus.

Note 76-D02: Changed the package composition

Up until now, the NITRO-System libraries, NITRO-Viewer, NITRO-SoundMaker and 3DTools were released as separate packages. Starting with this version, they are being released together as a single package.

NITRO-Viewer, NITRO-SoundMaker, and 3DTools are stored separately under the $NitroSystem/tools directory.

Note 76-D03: Added the Quick Start manual

Added the manual named Quick Start ($NitroSystem/docs/QuickStart.pdf), which explains the overview of setup when installing NITRO-System.
