Note 84-M01: Fixed a bug where intermediate files were not loaded properly on PCs set to certain languages

Fixed a bug where numeric data encoded in intermediate files was not loaded properly when the PC language was set to certain languages other than not Japanese or English.

Changes in 2.1.0

Note 81-M01: Fixed bug resulting in error when large element size in intermediate file

When the intermediate file contained elements with more than 2 MB of content, an error would occur. That has been fixed.

  • Changed the Select Object menu to the Object menu and changed the menu items.
  • Changed the way the object display position movement works in the object view schematic.
  • Added support for material settings' copy/paste functionality to multiple selected objects.
  • Deprecated ability to apply material settings to multiple other materials. (This is so that similar operations can be more intuitively performed by the addition of this functionality.)

Updated for NITRO intermediate file format version 1.5.0.

    • Added a depth buffering setting to the Camera property.
