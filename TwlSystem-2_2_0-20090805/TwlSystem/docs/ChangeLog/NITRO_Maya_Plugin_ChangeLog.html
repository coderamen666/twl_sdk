A bug where the Merge Useless Node process was not the same as the algorithm given in the manual has been fixed. Because the total number of nodes and hierarchical structure does not change, there is no need to re-output ICA files and IVA files.

Changes in Version 2004/02/12

Note 02-Y01: Changed such that all quadrilateral polygons in Maya are output as quadrilaterals

Quadrilateral polygons and convex quadrilateral polygons where all vertices are not in the same plane are no longer cut into triangular strips. Now quadrilateral polygons in Maya are all output as quadrilaterals. If hardware cannot display them properly, perform the appropriate triangle partitioning in Maya.

Note 02-Y02: Added support for changing the number of surfaces after skinning

Support for changing the number of surfaces after skinning has been added. Operations such as triangle partitioning after skinning is now possible. However, just as in the past, the number of vertices cannot be changed.

Note 02-Y03: Changed processing when the number of colors in a 4x4 texel compressed texture palette exceeds 32768

Revised such that image quality is lowered and palettes are compressed to 32768 colors or less when the number of colors of a palette for a 4x4 texel compressed texture exceeds 32768.

Changes in Version 2004/02/02

Note 01-Y01: Changed the effective number of lines for the Magnify and Tolerance export options (except Color)

The effective number of lines for the Magnify and Tolerance export options (except Color) has been changed up to the 10,000th-place digit.

Note 01-Y02: Fixed bug where Maya would crash depending on animation export conditions

A bug where Maya would crash when a texture SRT animation was set and another animation file was exported without outputting ITA has been fixed.
