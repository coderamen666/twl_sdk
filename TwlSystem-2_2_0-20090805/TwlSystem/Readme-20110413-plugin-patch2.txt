1. Introduction
==============================================================================

    This patch is a collection of update files for DCC tool plug-ins after TWL-System version 2009/08/05.

    To install this patch, copy the files into the directory where TWL-System version 2009/08/05 is installed. This will overwrite the existing files that have the same names.

　　This plugin-patch2 is a cumulative patch and as such includes revisions to plugin-patch.


2. File List
==============================================================================

　　Items updated with the TWL-System version 2011/04/13 plug-in patch 2
　　--------------------------------------------------------------------------
　　 Maya Plug-In
　　　 - Added support for Maya 2011
      - Ended support for Maya 8.5

　　 3ds Max Plug-ins
　　　- Added support for 3ds Max 2011
      - Ended support for 3ds Max 9.0


　　Items updated with the TWL-System version 2010/03/31 plug-in patch
　　--------------------------------------------------------------------------
　　Maya Plug-In
　　　- Added support for Maya 2010

　　Photoshop Plug-In
　　　- Updated the file format plug-in and the 4x4 texel compressed texture filter plug-in
　　　- Fixed a bug that created invalid 4x4 texel compressed textures.

　　SOFTIMAGE|XSI plug-in
　　　- Added support for Softimage 2010 SP1
　　　- Ended support for XSI 6.02
